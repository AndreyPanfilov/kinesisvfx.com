Neural Roto Assistant

We don't just simulate — we engineer perception

A Python script integrated into Nuke that automates rotoscoping of complex hair using PyTorch models.

Python Nuke AI PyTorch

Technical Overview

The Neural Roto Assistant was developed to solve the bottleneck of manual rotoscoping in high-volume VFX shots. By leveraging a custom-trained PyTorch model, the tool can predict hair edges with 95% accuracy, reducing manual labor by approximately 60%.

The pipeline integrates directly into the Nuke interface via a custom Python shelf tool. It utilizes OpenCV for pre-processing and passes masks directly to the node graph.
